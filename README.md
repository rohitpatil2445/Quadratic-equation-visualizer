# Quadratic Equation Visualizer

An interactive web application built with **Streamlit** that visualizes quadratic equations in real time. Users can modify the coefficients of the equation using sliders and instantly see how the graph changes.

## Features

- Interactive sliders for coefficients `a`, `b`, and `c`
- Real-time plotting of quadratic equations
- Simple and beginner-friendly interface
- Built using Streamlit and Matplotlib

## Equation

The application visualizes the quadratic equation:

y = ax² + bx + c

## Technologies Used

- Python
- Streamlit
- Matplotlib
- NumPy

## Installation

Clone the repository:

```bash
git clone <repository-url>
cd <repository-name>
```

Install the required libraries:

```bash
pip install streamlit matplotlib numpy
```

Run the application:

```bash
streamlit run app.py
```
